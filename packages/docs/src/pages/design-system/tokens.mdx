# Design Tokens

## Shadows

Smooth layered box shadows with a natural look.

Shadows are provided as CSS variables `--nn-shadow-lg` and `--nn-shadow-sm`.

    large
  </div>
  <div style="box-shadow: var(--nn-shadow-sm); padding: 1rem; border-radius: 1rem; margin: 3rem; width: 10rem; text-align: center; background: white;">
    small
  </div>
